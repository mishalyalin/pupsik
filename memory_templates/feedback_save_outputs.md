Always save generated files to `~/Desktop/claude/outputs/`.

**Why:** Avoid scattered output files across random directories. One predictable place for everything Claude produces.

**How to apply:** Whenever creating a file as output (report, draft, export, research doc, spreadsheet, PDF, HTML, etc.), save it to `~/Desktop/claude/outputs/` with a descriptive filename. Use date prefixes when appropriate (e.g. `2026-04-11-briefing.md`). This applies to ALL interfaces — Claude Code, Cowork, Desktop.
